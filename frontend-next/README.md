# Book-World Portfolio

A 2.5D side-scrolling creative portfolio inspired by *INSIDE* with a Naruto-themed "Book-World" aesthetic.

## Features
- **Proprietary Book Logic**: 3D page flips and Table of Contents (ToC) navigation.
- **Cinematic Environment**: Volumetric lighting, industrial fog, and paper grain textures.
- **Character Control**: GSAP-driven Naruto animations mapped to scroll position.
- **Responsive Architecture**: Smooth horizontal-to-vertical adaptation for mobile.

## Tech Stack
- React + TypeScript + Vite
- GSAP (ScrollTrigger)
- Tailwind CSS
- Lucide React
